# Prompt 場景庫 — AI 建築生成測試案例

> Claude Code 和使用者都可以用這些 prompt 測試系統
> 每個場景包含: prompt、預期輸出、AI 應自動處理的項目

---

## 住宅類

### 場景 1: 帶游泳池的3層別墅
```
Prompt: "帶游泳池的3層別墅"
土地: 150 坪

AI 自動處理:
- 搜尋別墅平面配置參考
- 1F: 客廳、廚房、餐廳、泳池區域
- 2F: 主臥套房、2間次臥、書房
- 3F: 家庭活動室、陽台
- 泳池尺寸約 8m × 4m，含機房
- 結構: RC 框架
- MEP: 含泳池循環水系統
```

### 場景 2: 12層住宅大樓
```
Prompt: "12層住宅大樓"
土地: 400 坪

AI 自動處理:
- 每層 4 戶，每戶 30 坪
- 地下 2 層停車場
- 2 部電梯 + 1 部緊急電梯
- 特別安全梯（15F 以下也建議設置）
- 耐震設計: 依工址震區係數
- 消防: 灑水系統、排煙、緊急照明
- 無障礙: 1F 坡道、無障礙電梯
```

## 商業/辦公類

### 場景 3: 帶地下停車場的商辦大樓
```
Prompt: "帶地下停車場的商辦大樓"
土地: 500 坪 商三用地

AI 自動處理:
- 地上 8 層辦公、地下 3 層停車
- 容積率/建蔽率合規
- 帷幕牆外觀
- 中央空調系統
- 每層獨立配電
- 消防: 排煙、灑水、緊急廣播
```

## 工業類

### 場景 4: 3層樓標準工廠廠房
```
Prompt: "3層樓標準工廠廠房"
土地: 1000 坪 丁種建築用地

AI 自動處理:
- 搜尋標準廠房規格
- 1F: 生產區（大跨距 15m+）、出貨區
- 2F: 輕加工區、倉儲
- 3F: 辦公區
- 貨梯 2 噸 × 1 部
- 客梯 × 1 部
- 工業通風系統
- 三相電力配置
```

### 場景 5: 100MW 數據中心
```
Prompt: "100MW 數據中心"
土地: 3000 坪

AI 自動處理:
- 搜尋 Tier 3/4 數據中心設計標準
- 伺服器大廳: 冷/熱通道配置
- 冷卻系統: 冷水機組 + 冷卻塔
- UPS 室 + 柴油發電機
- 雙路供電 + ATS
- 消防: 氣體滅火系統（FM-200 或 Novec）
- 安全: 門禁系統標注
- 樓板載重: 1000 kg/m² 以上
```

## 公共建築類

```
Prompt: "鄰近學校的社區活動中心"
土地: 200 坪

AI 自動處理:
- 1F: 多功能廳（可容 200 人）、辦公室
```

### 場景 7: 地區醫院
```
Prompt: "200 床地區醫院"
土地: 2000 坪

AI 自動處理:
- 搜尋醫院建築規範
- B2-B1: 停車場、機電室
- 1F: 急診、門診、藥局、大廳
- 2-3F: 手術室、ICU、檢查室
- 4-7F: 病房（每層 30 床）
- 8F: 行政辦公

- 特殊排水（醫療廢水）

```

## 使用方式

```bash
# CLI 測試
promptbim generate "帶游泳池的3層別墅" --land-area 150坪
promptbim generate "100MW 數據中心" --land-area 3000坪

# GUI 測試
# 1. 啟動 App
# 2. 在 Chat 面板貼上任一 prompt
# 3. 觀察 AI 自動生成過程
```
