# PromptBIM API Documentation

> Version: v2.0.0 | Updated: 2026-03-26

---

## Overview

PromptBIM provides a Python API for AI-powered BIM building generation. The core workflow is:

1. Define land parcel data
2. Set zoning rules
3. Run the Orchestrator pipeline
4. Get IFC + USD output files

---

## Quick Start

```python
from promptbim.agents.orchestrator import Orchestrator
from promptbim.schemas.land import LandParcel
from promptbim.schemas.zoning import ZoningRules

# 1. Define land
land = LandParcel(
    name="My Site",
    boundary=[(0, 0), (40, 0), (40, 30), (0, 30)],
    area_sqm=1200.0,
)

# 2. Set zoning rules
zoning = ZoningRules(city="Taipei")

# 3. Generate
orch = Orchestrator(output_dir="./output")
result = orch.generate("5-story residential building", land, zoning)

# 4. Use results
if result.success:
    print(f"Building: {result.building_name}")
    print(f"IFC: {result.ifc_path}")
    print(f"USD: {result.usd_path}")
```

## Core Schemas

### LandParcel

```python
from promptbim.schemas.land import LandParcel

parcel = LandParcel(
    name="Site A",
    boundary=[(0, 0), (50, 0), (50, 40), (0, 40)],  # meters, local coords
    area_sqm=2000.0,
    # Optional:
    address="Taipei, Xinyi District",
    zoning_code="R-3",
    ai_confidence=0.95,  # if parsed from image
)
```

### BuildingPlan

```python
from promptbim.schemas.plan import BuildingPlan

# Usually generated by PlannerAgent, but can be constructed manually:
plan = BuildingPlan(
    name="Villa",
    num_stories=3,
    story_height=3.2,
    footprint=[(0, 0), (20, 0), (20, 15), (0, 15)],
    spaces=[...],  # List of SpaceDef

```

```python
# Auto-loads BCR/FAR data for the city

```

## Agents

### Orchestrator (Pipeline Manager)

```python
from promptbim.agents.orchestrator import Orchestrator

def on_status(status):
    print(f"[{status.stage}] {status.message} ({status.progress:.0%})")

orch = Orchestrator(
    output_dir="./output",
    on_status=on_status,  # Optional progress callback
)

result = orch.generate(prompt, land, zoning)
```

### Individual Agents

Each agent can be used independently:

```python
# Enhancer: Expand user prompt into detailed requirements
from promptbim.agents.enhancer import EnhancerAgent
enhancer = EnhancerAgent()
requirements = enhancer.enhance("3-story villa")

# Planner: Generate building plan from requirements
from promptbim.agents.planner import PlannerAgent
planner = PlannerAgent()
plan = planner.plan(requirements, land, zoning)

# Builder: Generate IFC + USD files (no LLM, pure Python)
from promptbim.agents.builder import BuilderAgent
builder = BuilderAgent(output_dir="./output")
paths = builder.build(plan)

# Checker: Run compliance checks
from promptbim.agents.checker import CheckerAgent
checker = CheckerAgent()
report = checker.check(plan, land, zoning)

# Modifier: Apply natural language modifications
from promptbim.agents.modifier import ModifierAgent
modifier = ModifierAgent()
new_plan = modifier.modify(plan, "change to 9 floors")
```

## Land Parsers

```python
# GeoJSON
from promptbim.land.parsers.geojson import parse_geojson
parcels = parse_geojson("site.geojson")

# Shapefile
from promptbim.land.parsers.shapefile import parse_shapefile
parcels = parse_shapefile("site.shp")

# DXF
from promptbim.land.parsers.dxf import parse_dxf
parcels = parse_dxf("site.dxf")

# KML
from promptbim.land.parsers.kml import parse_kml
parcels = parse_kml("site.kml")

# PDF (OCR) - requires [pdf] extra
from promptbim.land.parsers.pdf_ocr import PDFLandParser
parser = PDFLandParser()
parcels = parser.parse("cadastral.pdf")

# AI Image Recognition - requires API key
from promptbim.land.parsers.image_ai import parse_land_image
parcel = parse_land_image("aerial_photo.jpg")
```

## BIM Generators

### IFC Generator

```python
from promptbim.bim.ifc_generator import IFCGenerator

gen = IFCGenerator()
ifc_path = gen.generate(plan, output_dir="./output")
# Output: ./output/building.ifc
```

### USD Generator

```python
from promptbim.bim.usd_generator import USDGenerator

gen = USDGenerator()
usd_path = gen.generate(plan, output_dir="./output")
# Output: ./output/building.usda

# USDZ packing
from promptbim.bim.usdz_packer import pack_usdz
usdz_path = pack_usdz(usd_path)
```

## Code Compliance

```python
from promptbim.codes.registry import run_all_checks, get_compliance_summary
```

## MCP Server

The MCP server exposes PromptBIM functionality to Claude Desktop:

```bash
# Start server
python -m promptbim.mcp.server

# Or configure in Claude Desktop config:
# See mcp/config.json for setup
```

**Available MCP Tools:**
- `generate_building` - Generate from prompt
- `check_compliance` - Run code checks
- `estimate_cost` - Cost estimation
- `import_land` - Parse land file
- `list_templates` - Available templates
- `health_check` - System status
- `modify_building` - Apply modifications

## Configuration

All settings are loaded from `.env` file:

```env
ANTHROPIC_API_KEY=sk-ant-api03-...
CLAUDE_MODEL=claude-sonnet-4-20250514
OUTPUT_DIR=./output
DEFAULT_CITY=Taipei
DEBUG_MODE=false
STARTUP_CHECK_ENABLED=true
```

Programmatic access:

```python
from promptbim.config import get_settings

settings = get_settings()
print(settings.claude_model)
print(settings.default_city)
```
