## 建築高度限制

```
建築高度 ≤ 前面道路寬度 × 1.5 + 6m

範例:
  道路寬 6m  → 高度上限 15m（約 4-5 層）
  道路寬 8m  → 高度上限 18m（約 5-6 層）
  道路寬 10m → 高度上限 21m（約 6-7 層）
  道路寬 15m → 高度上限 28.5m（約 8-9 層）
  道路寬 20m → 高度上限 36m（約 10-12 層）
  道路寬 25m → 高度上限 43.5m（約 12-14 層）
```
